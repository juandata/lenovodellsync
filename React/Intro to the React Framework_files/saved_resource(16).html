            Daniel15 4 years ago

            Great tutorial. Just a few little points:
            1. This:
            console.log(this.refs.counter.innerHTML);

            Should be this:
            console.log(this.refs.counter.getDOMNode().innerHTML);

            2. This:
            var shapeTree = <div class="shape-canvas">
            {
            this.props.shapes.map(function(s) {
            return self._createShape(s);
            })

            Can be simplified to this:
            var shapeTree = <div class="shape-canvas">
            {this.props.shapes.map(this._createShape)}

            When doing a .map() you don't need to keep a reference to 'this', you can just pass it in as the second parameter. If the function takes one parameter, you can just pass it in directly (like I did here).
